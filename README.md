# 🔑 Token Secret Generator

> *karena ngetik random string sendiri itu tidak se-random yang kamu kira*

---

Buka browser. Klik generate. Dapat token kriptografis yang proper. Selesai.

Tidak ada server. Tidak ada database. Tidak ada yang ngintip. Semua jalan di browser kamu sendiri — bahkan bisa dipakai offline.

---

## ✨ ada apa aja?

- **Token utama** — hex, base64, base64url, atau UUID v4, terserah
- **Turunan otomatis** — JWT secret, HMAC key, AES-256 key, AES IV, API key, SHA-256 hash
- **Export `.env`** — langsung copy-paste ke project, beres
- **Batch mode** — butuh 20 token sekaligus? bisa
- **Shortcut** `Ctrl+Enter` — buat yang males klik tombol
